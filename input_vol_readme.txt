硬體注意事項:

    電壓分壓器:

        對於0-100V輸入，使用電壓分壓器(例如95kΩ + 5kΩ)將100V縮小到5V供ADC使用

        根據實際電阻值調整VOLTAGE_DIVIDER_RATIO

    7段顯示器:

        此代碼假設使用共陰極顯示器

        調整段和數位引腳定義以匹配您的硬體連接

        可能需要為每個段添加限流電阻(通常220Ω)

    ADC參考電壓:

        確保穩定的5V參考電壓以獲得準確測量

        考慮在ADC輸入引腳添加小電容(0.1μF)

工作原理:

    ADC測量按比例縮小的輸入電壓

    測量值轉換為實際電壓(0-100V範圍)

    電壓格式化為顯示格式(XX.X格式)

    定時器中斷處理顯示多工以顯示所有3位數

    小數點自動放在第二位數之後

自定義:

    如果使用不同的電阻，調整電壓分壓器比例

    更改段/數位引腳分配以匹配您的硬體

    如果需要不同的精度，修改顯示格式
